## 🚀 Installation

- Open the Extensions sidebar in VS Code
- Search for `Vivid Black`
- Click Install
- Open the Theme Palette with Ctrl+K + Ctrl+T or ⇧ k + ⇧ T
- Color Theme and choose `Vivid Black`.
